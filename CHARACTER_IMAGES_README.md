# Инструкция по добавлению изображений персонажей

## Структура папок

Все изображения должны находиться в папке `public/`:

```
public/
├── start-background.jpg          # Фон для стартового экрана
├── characters/
│   ├── player_067.png           # Полное изображение персонажа Player 067
│   ├── player_456.png           # Полное изображение персонажа Player 456
│   ├── player_218.png           # Полное изображение персонажа Player 218
│   ├── player_199.png           # Полное изображение персонажа Player 199
│   ├── default.png              # Запасное изображение
│   └── avatars/
│       ├── player_067_avatar.png  # Круглая аватарка Player 067
│       ├── player_456_avatar.png  # Круглая аватарка Player 456
│       ├── player_218_avatar.png  # Круглая аватарка Player 218
│       └── player_199_avatar.png  # Круглая аватарка Player 199
```

## Требования к изображениям

### Стартовый экран
- **start-background.jpg**: Фоновое изображение для стартового экрана
  - Рекомендуемый размер: 1920x1080px
  - Формат: JPG или PNG

### Персонажи (полный рост)
- **characters/player_XXX.png**: Изображения персонажей в полный рост
  - Рекомендуемый размер: 500x800px (портретная ориентация)
  - Формат: PNG с прозрачным фоном
  - Персонаж должен быть по центру
  - Используется на экране выбора персонажа

### Аватарки (круглые)
- **characters/avatars/player_XXX_avatar.png**: Круглые аватарки персонажей
  - Рекомендуемый размер: 128x128px (квадрат)
  - Формат: PNG
  - Изображение должно быть круглым или хорошо смотреться в круглой рамке
  - Используется в лобби, в игре, в списке игроков

## Как добавить изображения

1. Подготовьте изображения согласно требованиям выше
2. Поместите их в соответствующие папки в `public/`
3. Убедитесь, что имена файлов совпадают с указанными
4. Перезапустите сервер разработки

## Запасной вариант

Если изображение не найдено, система автоматически:
- Для персонажей: покажет силуэт с градиентным свечением
- Для аватарок: покажет текстовый ID персонажа

## Пример добавления своего персонажа

Чтобы добавить нового персонажа:

1. Откройте `data/characters.ts`
2. Добавьте новый объект в массив `CHARACTERS`:

```typescript
{
  id: 'player_999',
  name: 'PLAYER 999',
  description: 'Ваше описание персонажа',
  speed: 80,
  damage: 70,
  health: 90,
  imagePath: '/characters/player_999.png',
  avatarPath: '/characters/avatars/player_999_avatar.png',
}
```

3. Добавьте соответствующие изображения в папку `public/characters/`
